# ShopSmart-Ecommerce-MinorProject-
## Build and end-to-end supervised ML pipeline using Random Forest , Decision Tree , Logistic regression , Gaussian Naive Bayes to Predict Visitor Behaviour.Implemented Binary classification along with EDA , feature engineering & model evalution accuracy , recall ,f1 

### 🔎 A predictive model that can determine whether a visitor is likely to make a purchase based on their session behaviour.

## ⏬Model desciprtion:
Dataset contain 12,330 individual with 18 features

# ⬇️Model transformation with :
1. Handling Null values
2. EDA
3. Encoding - OneHotEncoder , LabelEncoder
4. Scaling - StandardScaler
5. Feature Selection
6. Model Selection - train_test_split , GridSearchCV
7. Pipeline for preventing DataLeakage
8. Multiple HyperTuning with RandomForest , DecisionTree , LogisticsRegression ,Gaussian Naive bayes

## ➡️Algorithms Used:
1. LogisticRegression
2. DecisionTree
3. RandomForest
4. Gaussian Naive Bayes

## 🔽 Library Used:
1. scikit Learn
2. Pandas
3. Numpy
